下列与《交付工期与清单_5月1日启动》一致，本页仅作可见性分层。绿色行 = 业主/客户能直接感知的系统能力或界面。

交付物 类型标识 说明
Web 管理后台（构建产物、访问方式） 管理/项目端可见 RFP「+ 网页管理后台」：排班、任务、问题、材料/TOC、AI 结果、权限；不默认对业主开放账号。
项目专属 H5/网站 + 客户视图 终用户可见 二-4 客户远程观测：日/周更新、照片+说明、评论提问、节点确认；权限脱敏后对外。
微信小程序（与 H5/网站同源 API） 终用户可见 满足 RFP「三、移动端：微信小程序或 App」中的 小程序 线；含发版/体验码/提审材料（以微信审核周期为准，不阻塞后端验收）。
工程内聊天（按项目频道） 终用户可见 与进度同入口或独立 Tab；@任务/问题视实现为准。
实时推送（WSS / SSE 展示进度/消息变化） 终用户可见 页面「自动刷新/待办数变化」等，属于客户/管理端可感知体验。
专属 Android APK + 安装说明页/二维码 终用户可见 RFP「或 App」线：安装后与网站/小程序 同源能力（按基线范围）。
多模态分析结果（在任务/影像详情中的解读区） 管理/项目端 / 若对客户开放则 终用户可见 默认在管理端最完整；是否对客户展示「摘要/结论」由基线定。
后端包/镜像、DB 初始化、对象存储、网关路由说明 随验收移交 给甲方技术/信息部门；不对业主直晒。
《系统功能说明》《操作手册》《测试/验收报告》等 随验收移交 可复印给各角色；内容含客户操作章节即属「用户能照着用」的交付。
代码仓库/压缩包、线框与 Figma 链接、接口文档 仅内部/研发 二开/审计用，业主一般不接触。
